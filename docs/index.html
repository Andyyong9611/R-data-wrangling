Prerequisites and Preparations

  • You should have some basic knowledge of R, and be familiar with the topics covered in the Introduction to R.
  • Have a recent version of R and RStudio installed.
  • Install and load the tidyverse package.
  • Download MS_trafficstops_bw.csv from here: https://github.com/cengel/R-data-wrangling/raw/master/data/MS_trafficstops_bw_age.csv
  • Recommended: Create a new RStudio project R-data-ws in a new folder R-data-ws and download MS_trafficstops_bw.csv into a subdirectory called data.
